INTRODUCTION OF HTML

Abstract: HTML is the Standard Markup Language. It is used for developing Web Pages. HTML is Hyper Text Markup Language and is used for describing the structure of web pages. Various Tags are used in HTML like "heading", "paragraph", "table", and so on. This paper discusses various HTML tags that are must for developing web pages.
